• Windows is het stuurprogramma dat wij gebruiken, naast Windows gebruiken wij ook Microsoft 365
  • Microsoft 365
    Microsoft 365 is een verzameling van Windowsapplicaties zoals: Word, Excel, Outlook en Teams. Dit is software die je nodig hebt om te kunnen werken. Microsoft 365 wordt gratis aangeboden door de school!
  • Visual Studio Code is het programma waar wij in programmeren met HTML, CSS, Javascript, PHP en Python
  • GitHub word gebruikt om de programmeer scripts op te slaan in de Cloud en te delen met andere mensen en docenten
  • Voor deze opleiding heb je een laptop nodig die aan deze eisen moeten voldoen.
  • Beeldscherm van minimaal 15 inch.
  • Minimaal Intel Core i5 of Ryzen 5.
  • Windows 10 besturingssysteem (of nieuwer).
  • 8GB ram (niveau 2) en 16GB ram (niveau 3 en 4).
  • SSD schijf van minimaal 256 GB
  • Netwerkaansluiting om bekabeld verbinding te kunnen maken. (Alternatief: een usb naar utp adapter).
